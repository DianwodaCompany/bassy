# 安装部署
1.在bassy-mapper模块下，存放有数据库表和数据初始化sql文件，请先初始化数据库信息。  
2.在bassy-web模块下，配置数据库和redis的连接配置。  
3.在bin目录下，执行install.sh脚本，会自动打包并启动服务。
